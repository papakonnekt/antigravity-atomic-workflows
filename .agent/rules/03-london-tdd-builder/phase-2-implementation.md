> **Objective**: Make the test pass. Minimum Viable Code.

## Prerequisites
**MUST be completed before this phase**:

## Instructions

### Step 1: Create Source File
**Action**: Create or update the source file `src/[feature].ts`.

### Step 2: Autonomy Check
- **If Manual**: Ask user "Proposed Code: [Summary]. Shall I write this?" before creating source file.
- **If Auto/Semi**: Proceed.

### Step 3: Write Code
**Action**: Write the *minimum* code required to pass the test. Do not over-engineer.

### Step 4: Run Test
**Action**: Execute `npm test`.

### Step 5: The Green Loop
- **If Red**: Read the Error Message. Debug. Fix. Loop.
- **If Green**: Proceed to exit phase.

### Step 6: Phase Checklist
**Action**: Open `.agent/checklists/03-london-tdd-builder/phase-2-implementation-checklist.md`.
**Instruction**: Verify every item is checked.

---

## Exit Criteria
**This phase is complete when ALL of the following are true**:
- ✅ Source code written.
- ✅ Test passes (Green state verified).
- ✅ Checklist verified.
