# Phase 1: Job Hunt & Context Absorption

> **Parent Workflow**: 02-SPARC-Specification
> **Phase Number**: 1
> **Objective**: Find the next Ready-for-Spec feature.

---

## Prerequisites
**MUST be completed before this phase**:
- ✅ Phase 0: The Ancestral Audit

---

## Instructions

### Step 1: Context Loading
**Action**: Read `memory/PROJECT_STATE.md` and `memory/KNOWLEDGE_BASE.md`.

### Step 2: Scan
**Action**: Find the **First** feature with status `(R)` (Researched) that is NOT yet `(S)`.

### Step 3: Deep Dive
**Action**: Read the corresponding `docs/research/[feature]/deep_dive.md`.
**Action**: Read the `docs/architecture/system_architecture_manifest.md`.

### Step 4: Staging
**Action**: Create directory `memory/openspec/changes/[feature]/specs/`. (This is the **Proposal Staging Area**).

### Step 5: Phase Checklist
**Action**: Open `.agent/checklists/02-sparc-specification/phase-1-job-hunt-checklist.md`.
**Instruction**: Verify every item is checked.

---

## Exit Criteria
**This phase is complete when ALL of the following are true**:
- ✅ Feature selected.
- ✅ Context files read.
- ✅ Proposal directory created.
- ✅ Checklist verified.
