# Phase 6: Pseudocode Generation & Logic Linting

> **Parent Workflow**: 02-SPARC-Specification
> **Phase Number**: 6
> **Objective**: Write the logic and SIMULATE it.

---

## Prerequisites
**MUST be completed before this phase**:
- ✅ Phase 5: Functional Specification

---

## Instructions

### Step 1: Literate Programming
**Action**: Write the logic for the *Main* function/component in plain English.

### Step 2: Mental Sandbox (Logic Linting)
**SCoT**: "Simulating the code run..."
- **Simulation**: Run the pseudocode in your "Mind's Eye" against the Phase 4 Edge Cases.
- **Check**: "If I pass NULL to this pseudocode logic, does it handle it?"
- **Correction**: Fix the logic **before** writing the description.

### Step 3: Action
**Action**: Write `memory/openspec/changes/[feature]/specs/pseudocode.md`.

### Step 4: Phase Checklist
**Action**: Open `.agent/checklists/02-sparc-specification/phase-6-pseudocode-checklist.md`.
**Instruction**: Verify every item is checked.

---

## Exit Criteria
**This phase is complete when ALL of the following are true**:
- ✅ pseudocode.md created.
- ✅ Logic validated against edge cases.
- ✅ Checklist verified.
