# Phase 3.5: Reflection Checkpoint
> **Parent Workflow**: 02-SPARC-Specification
> **Phase Number**: 3.5
> **Objective**: Self-critique and course-correct before proceeding.

---

## Prerequisites
**MUST be completed before this phase**:
- ✅ Phase 3: The Devil's Advocate Interceptor

---

## Instructions

### Step 1: The Pause
**Action**: Stop all action and review all outputs generated so far.

### Step 2: Self-Critique Questions
**Action**: Answer the following:
1.  "What could be wrong with my current approach?"
2.  "What information am I missing?"
3.  "Am I making any assumptions I haven't verified?"

### Step 3: Gap Identification
**Action**: List any identified gaps. For each gap, decide if additional research is needed.

### Step 4: Decision
**Action**:
- **No gaps**: Proceed to next phase.
- **Minor gaps**: Log in `MEMORY_STREAM.md`, proceed with caution.
- **Major gaps**: Loop back to Phase 1 (max 2 times).

### Step 5: Log Outcome
**Action**: Write reflection outcome to `MEMORY_STREAM.md`:
```markdown
## Reflection Checkpoint [Date]
- Gaps Found: [List]
- Action Taken: [Proceeded/Looped Back]
- Confidence Level: [High/Medium/Low]
```

### Step 6: Phase Checklist
**Action**: Open `.agent/checklists/02-sparc-specification/phase-3.5-reflection-checklist.md`.
**Instruction**: Verify every item is checked.

---

## Exit Criteria
**This phase is complete when ALL of the following are true**:
- ✅ Reflection logged and decision made.
- ✅ Checklist verified.
