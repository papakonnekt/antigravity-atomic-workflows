> **Parent Workflow**: 04-bmo-triangulation
> **Phase Number**: 4.8
> **Objective**: Generate self-critique and extract lessons for Episodic Memory.

---

## Prerequisites
**MUST be completed before this phase**:
- ✅ The main work of the workflow.

---

## Instructions

### Step 1: Self-Critique Generation
**Action**: Answer the following questions in `memory/MEMORY_STREAM.md`:
1.  "What would I do differently next time?"
2.  "What assumptions were incorrect?"
3.  "What worked particularly well?"

### Step 2: Lesson Extraction
**Action**: Extract 1-2 actionable lessons from the critique.
**Format**: "In [situation], [action] led to [outcome]. Next time: [adjustment]."

### Step 3: Episodic Memory Update
**Action**: Add the Lesson Learned to `memory/EPISODIC_MEMORY.md`.
**Action**: Tag with: feature type, workflow name, purity score.

### Step 4: Dashboard Update
**Action**: Update `memory/HEALTH_DASHBOARD.md` with:
- Loops used in this session.
- Final Mirror Test score.
- Anti-patterns detected.

### Step 5: Phase Checklist
**Action**: Open `.agent/checklists/04-bmo-triangulation/phase-4.8-reflexion-checklist.md`.
**Instruction**: Verify every item is checked.

## Exit Criteria
**This phase is complete when ALL of the following are true**:
- ✅ Critique logged.
